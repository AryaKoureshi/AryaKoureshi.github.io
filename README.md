# Arya Koureshi — Portfolio

Personal portfolio for [Arya Koureshi](https://aryakoureshi.github.io), Senior AI Engineer and Full-Stack Developer.

The homepage is a lightweight static site built with semantic HTML, custom CSS, and a small amount of vanilla JavaScript. Legacy Jekyll project and CV routes remain available for existing links.

## Local preview

From the repository root:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Main content

- Product work at Done Institute: DoneProject, DoneGram, and Daramhavato
- Selected AI, computer vision, medical imaging, and EEG/BCI projects
- Research, teaching, education, contact information, and downloadable resume

The site is deployed automatically by GitHub Pages from the `master` branch.
